## ScalaFX-Extras Release v.0.12.0

This is a feature release of [ScalaFX-Extras].

### New features

* ImageDisplay: add ability to apply Effect to displayed image [[#40]]
* ScalaFX updated to v.25.0.2-R37

### API Breaking Changes

* ImageDisplay: Expose "rotation" as a property [[#42]]]
  - before `rotation` was not observable extenally

### Scala 3 support

* Focus Scala sources on Scala 3 LTS [[#41]]  
  - drop support for Scala 2
